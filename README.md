# Livechat One Monitor Premium

A transparent, always-on-top **live-chat overlay** for streamers. It pulls live
chat from **YouTube, Twitch, and TikTok** at the same time, shows **Trakteer
donations**, can read chat aloud with **Text-to-Speech**, and pushes a
fully-styled chat overlay straight into **OBS**.

This repository hosts the **official installer and updates** only. The source
code is not public.

## ✨ Features

- **Multi-platform chat** — YouTube, Twitch & TikTok live chat together, with
  platform badges and a Collab mode (host + 2 guests).
- **OBS overlay** — a fully customizable chat overlay (colors, gradients, box
  images, fonts, 20+ entrance animations, particles, per-role colors). One-click
  "Add to OBS".
- **Trakteer donations** — donation alerts with gift media (GIF/video) and voice
  messages.
- **Text-to-Speech** — reads chat aloud (free online voices or your own offline
  custom voices).
- **Streamer Mode** — hide the control window from OBS/screen capture.
- **Performance Mode** — lighten the load on weaker PCs.
- **Game auto-lock** — makes the overlay click-through automatically while gaming.
